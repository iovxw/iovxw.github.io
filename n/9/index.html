顺便给博客添加搜索功能

2014-06-20 00:00

谁说静态博客就不能用搜索的？

好吧其实就是用了下搜索引擎的site搜索功能

可以看见右边测栏里已经多出来了一个搜索框

不过貌似百度没有收录这个博客= =

所以没法用什么的很正常

在页面内添加

<form id="search" class="search" role="search">
        <input class="form-control" placeholder="搜索" name="q" type="text">
</form>

然后在js里添加

(function(){
	$(document).on('submit', '#search.search', function(){
		var q = $(this).find('[name=q]').val();
		if($.trim(q) !== ''){
			var host = window.location.hostname;
			var url = 'http://www.baidu.com/s?wd=' + 'site:' + host + " " + $.trim(q);
			window.open(url, "_blank");
		}
		return false;
	});
})();

http://www.baidu.com/s?wd=可以替换成任意支持site搜索的搜索引擎地址

PS: 这是 jQuery 版本，写成原生的也不难这里就不废话了

  29.     fmt.Printf("%v: %v\n", err, desc)

  32. func main() {
  33.     glfw.SetErrorCallback(errorCallback)
  35.     if !glfw.Init() {
  36.         panic("Can't init glfw!")
  37.     }

  64. func createTexture(r io.Reader) (gl.Texture, error) {
  65.     img, err := png.Decode(r)
  66.     if err != nil {
  67.         return gl.Texture(0), err
  68.     }
  70.     rgbaImg, ok := img.(*image.NRGBA)
  71.     if !ok {
  72.         return gl.Texture(0), errors.New("texture must be an NRGBA image")
  73.     }
  75.     textureId := gl.GenTexture()
  76.     textureId.Bind(gl.TEXTURE_2D)
  77.     gl.TexParameterf(gl.TEXTURE_2D, gl.TEXTURE_MAG_FILTER, gl.LINEAR)
  78.     gl.TexParameterf(gl.TEXTURE_2D, gl.TEXTURE_MIN_FILTER, gl.LINEAR)
  80.     // flip image: first pixel is lower left corner
  81.     imgWidth, imgHeight := img.Bounds().Dx(), img.Bounds().Dy()
  82.     data := make([]byte, imgWidth*imgHeight*4)
  83.     lineLen := imgWidth * 4
  84.     dest := len(data) - lineLen
  85.     for src := 0; src < len(rgbaImg.Pix); src += rgbaImg.Stride {
  86.         copy(data[dest:dest+lineLen], rgbaImg.Pix[src:src+rgbaImg.Stride])
  87.         dest -= lineLen
  88.     }
  89.     gl.TexImage2D(gl.TEXTURE_2D, 0, 4, imgWidth, imgHeight, 0, gl.RGBA, gl.UNSIGNED_BYTE, data)
  91.     return textureId, nil
  92. }
  94. func initScene() (err error) {
  95.     gl.Enable(gl.TEXTURE_2D)
  96.     gl.Enable(gl.DEPTH_TEST)
  97.     gl.Enable(gl.LIGHTING)

让XP继续接受更新

通过修改注册表来让XP继续接受更新

2014-05-27 00:00

在Betanews看到了让XP继续更新的方法

大体过程如下：

1.新建一个文件扩展名为.reg的文件（例如updates.reg）

2.用文本编辑器打开

3.修改内容为

Windows Registry Editor Version 5.00
[HKEY_LOCAL_MACHINE\SYSTEM\WPA\PosReady]
"Installed"=dword:00000001

4.保存，然后双击它

现在打开更新管理器就会发现有新的更新可用了

这个的原理是欺骗Windows Update，让它把你的电脑识别为Windows Embedded POSReady 2009

而Windows Embedded POSReady 2009的支持日期一直到2019年4月9日

当然这不是最好的解决办法，要想一直有免费升级的话，还是去用Linux吧

最起码你不用再担心系统是不是安全了
